出坑记录

导入数据时有可能会报错
原因是 mysql 默认没有开 secure_file_priv（这个参数用来限制数据导入和导出操作这些操作需要用户具有FILE权限。 ）

报错如下:

解决办法：在mysql的配置文件中（my.ini 或者 my.conf）中添加 secure_file_priv = /Users/Hoe/temp/, 然后重启 mysql
