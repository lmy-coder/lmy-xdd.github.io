Hadoop安装

发表于 2018-07-21 分类于 机器

find -name mysql 查看Linux里面有没有MySQL （find -name查找什么后面就写什么）

Hadoop的文件路径 cd / opt hadoop-2.7.2 hadoop

安装 Hadoop

首先安装ssh

CentOS的软件安装工具不是apt-get 是yum

yum -y install ssh openssh-server

1.创建ssh-key，这里我们采用rsa方式，使用如下命令：　

　　　ssh-keygen -t rsa -P “”

2.出现一个图形，出现的图形就是密码，不用管它　

　　　cat ~/.ssh/id_rsa.pub >> authorized_keys(好像是可以省略的)

　3.然后即可无密码验证登录了，如下：　

　　　ssh localhost

wget https://www.apache.org/dyn/closer.cgi/hadoop/common/hadoop-3.1.2/hadoop-3.1.2-src.tar.gz

把本地安装包传进Linux里的opt文件夹

在本地往Linux里传文件，在黑窗口里执行命令：scp 路径 root@192.168.200.134:/opt在Linux里要存的文件夹

使用如下命令解压缩Hadoop安装包

tar -zxvf hadoop-3.1.2.tar.gz

解压jbk

　解压缩完成后出现hadoop2.7.1的文件夹

解压完成后进入jbk的文件夹

然后配置vim ~/.bashrc 这个文件夹是隐藏的进去后输入i是编辑

export PATH=”/root/anaconda3/bin:$PATH”

export PATH=”/root/anaconda3/bin/python:$PATH”

export JAVA_HOME=”/opt/jdk1.8.0_60”

export PATH=”$PATH:$JAVA_HOME/bin”

export HADOOP_HOME=”/opt/hadoop-2.7.2”

export PATH=”$PATH:$HADOOP_HOME/bin:$HADOOP_HOME/sbin”

然后按ESC SHIFT； qw 是保存退出 q！不保存强制退出

进入文件夹 /opt/etc

vim hosts 填写master

vim hostname 填写127.0.0.1 master
